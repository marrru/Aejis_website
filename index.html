We Are

We develop our projects day by day, during the years, since 2009. And we know how to make continuous development possible. Software development is our passion, so we care a lot of the quality of our work. That's why we have adopted the strict quality standards.

Our Passion

It doesn't matter how big and complex your project is — we will take care of its maintainability.By Maintaining code review, test coverage, static code analysis, code documentation and pixel perfection we are reaching your absolute happiness.

Our Process

We build our process on the SCRUM framework with strong TDD approach. Each feature turns to the pull request, which should satisfy continuous integration service, code metrics service and at least three attentive reviewers. Each release goes to production through Kerberos QAs.

Our Philosophy

As soon as we use a lot of open source technologies we also are involved in open source community and try to enhance open source tools as much as we can. Our stack includes, but not limited to: Ruby, Rails, Sinatra, PostgreSQL, PostGIS, Neo4j, JavaScript, BackboneJS, MarionetteJS, SCSS, Objective-C, Swift, ReactiveCocoa, etc.
